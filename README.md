# 🎓 Student Management System

A simple and interactive Student Management System built using :contentReference[oaicite:0]{index=0} and :contentReference[oaicite:1]{index=1}.

This project allows users to manage student records, calculate grades, search students, and delete student data through an easy-to-use web interface.

---

## 🚀 Features

- Add Student Records
- View All Students
- Search Student by Roll Number
- Delete Student Record
- Automatic Grade Calculation
- Interactive Streamlit UI

---

## 🛠️ Tech Stack

- Python  
- Streamlit

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/student-management-system.git
```

Go to the project directory:

```bash
cd student-management-system
```

Install dependencies:

```bash
pip install streamlit
```

---

## ▶️ Run the Project

```bash
streamlit run app.py
```

### Search Student

## 🎯 Grade Calculation System

| Marks Range | Grade |
|---|---|
| 90 - 100 | A+ |
| 80 - 89 | A |
| 70 - 79 | B |
| 60 - 69 | C |
| 50 - 59 | D |
| Below 50 | Fail |
